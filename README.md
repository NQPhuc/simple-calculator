# simple-calculator

STEP FOR RUNNING ON LOCAL

A. Setup BE:
  1. change working directory to ./packages/api-server
  3. yarn db:init
  4. change sample.env name into .env
  5. start the server with: yarn dev

B. Setup FE
  1. change working directory to ./packages/client
  2. yarn install
  4. change sample.env name into .env
  5. start the server with: yarn dev
